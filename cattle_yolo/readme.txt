================== 新功能：数据库与可视化 ==================

1. 新增功能：
   - SQLite数据库存储：检测到的牛行为会自动保存到本地数据库（cattle_detection.db）
   - ECharts数据可视化：提供丰富的图表展示检测数据

2. 新增依赖：
   项目已更新requirements.txt，新增以下依赖：
     - pandas：数据处理
     - pyecharts：图表生成
     - streamlit-echarts：在Streamlit中显示ECharts图表

   请运行以下命令安装新依赖：
     pip install -r requirements.txt

3. 使用方法：
   - 启动应用后，在侧边栏"选择要进行的任务"中可选择：
     1. "目标检测"：原有的图片/视频/摄像头检测功能
     2. "数据可视化"：查看检测数据统计和图表

   - 在"数据可视化"页面中，您可以：
     * 选择日期范围查看数据
     * 查看行为分布饼图
     * 查看每日检测趋势图
     * 查看行为统计柱状图
     * 查看检测来源分析
     * 浏览最近检测记录并下载CSV数据

4. 数据库说明：
   - 数据库文件：cattle_detection.db（自动创建）
   - 表结构：detection_log（检测记录表）
   - 字段：时间戳、源类型、源名称、行为、置信度、位置等

注意：所有检测操作（图片、视频、摄像头）都会自动记录到数据库，您可以在可视化页面查看历史数据。
